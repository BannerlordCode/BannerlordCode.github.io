# Bannerlord v1.4.5

## Mental Model

Treat `Bannerlord v1.4.5` as an entry point or data node for this subsystem: inspect its properties first, then decide which methods to call.

::: tip Source now available
Decompiled v1.4.5 source is organized in `bannerlord-1.4.5/Bannerlord.Source/`: gameplay modules (SandBox / StoryMode / Multiplayer / CustomBattle / FastMode / BirthAndDeath / Native extensions) total 2361 .cs files / 2523 types; core `TaleWorlds.*` assemblies ship as DLLs in `bin/`. Start from the API catalog, then follow each class page's breadcrumb back here.
:::

## Module list

v1.4.5 ships **65 `TaleWorlds.*` managed assemblies**. Compared to v1.3.15:

- **14 new**: `CampaignSystem.FastMode`, `CampaignSystem.ViewModelCollection.BirthAndDeath`, `MountAndBlade.CustomBattle`, `MountAndBlade.DedicatedCustomServer.ClientHelper`, `MountAndBlade.GauntletUI`, `MountAndBlade.GauntletUI.AutoGenerated.0/.1`, `MountAndBlade.Multiplayer`(+`.GauntletUI`/`.View`/`.ViewModelCollection`/...), `MountAndBlade.Platform.PC`, `MountAndBlade.View`
- **4 removed**: `MountAndBlade.GauntletUI.CodeGenerator`, `MountAndBlade.Launcher`, `MountAndBlade.SaveSystem.CodeGenerator`, `MountAndBlade.SteamWorkshop`
- **Gameplay modules return**: `SandBox`(+.GauntletUI/.View/.ViewModelCollection) and `StoryMode`(+...), missing from the 1.3.15 source bundle, are provided as source in 1.4.5.

See v1.3.15 [Version Delta](../../v1.3.15/en/architecture/version-delta) and [SDK Overview](../../v1.3.15/en/architecture/sdk-overview).
