# Architecture Overview

**Namespace**: TaleWorlds.*
**Version**: 1.3.15 (compared with 1.3.0)

## Module Hierarchy

Bannerlord consists of 47 TaleWorlds.* modules forming this dependency hierarchy:

```
TaleWorlds.Library (Base utilities)
    │
    ├── TaleWorlds.LinQuick (LINQ optimizations)
    ├── TaleWorlds.Localization (Localization)
    └── TaleWorlds.InputSystem (Input handling)
            │
            ▼
TaleWorlds.Core (Core: items, characters, crafting)
    │
    ├── TaleWorlds.ObjectSystem (XML deserialization)
    ├── TaleWorlds.DotNet (.NET interop)
    └── TaleWorlds.ScreenSystem (Screen management)
            │
            ▼
TaleWorlds.Engine (Native engine bridge)
    │
    ├── TaleWorlds.Engine.AutoGenerated
    └── TaleWorlds.NavigationSystem (Navigation)
            │
            ▼
TaleWorlds.MountAndBlade (Game engine core)
    │
    ├── TaleWorlds.CampaignSystem (Campaign system)
    ├── TaleWorlds.MountAndBlade.ViewModelCollection
    └── TaleWorlds.MountAndBlade.AutoGenerated
            │
            ▼
TaleWorlds.ModuleManager (Module management)
TaleWorlds.SaveSystem (Save system) ← NEW in 1.3.15
```

## 1.3.15 vs 1.3.0 Comparison

| Stat | 1.3.15 | 1.3.0 | Difference |
|------|--------|-------|------------|
| TaleWorlds modules | 47 | 23 | +24 |
| Core module | TaleWorlds.MountAndBlade | Same | No change |
| New module | TaleWorlds.SaveSystem | ❌ | ★ NEW |
| UI system | GauntletUI (enhanced) | GauntletUI | Major update |

### New Modules in 1.3.15

- **TaleWorlds.SaveSystem** - Brand new save system
- **TaleWorlds.PSAI** - AI system
- **TaleWorlds.ActivitySystem** - Activity system
- **TaleWorlds.Diamond** - Network multiplayer system refactored
- **TaleWorlds.PlayerServices** - Player services
- **TaleWorlds.PlatformService.*** - Platform services (Steam/Epic/GOG)
- **TaleWorlds.ServiceDiscovery.Client** - Service discovery

## Core Entry Points

| Class | File | Responsibility |
|-------|------|----------------|
| `MBSubModuleBase` | MBSubModuleBase.cs | Base class for every mod |
| `Module` | Module.cs | Main module, controls entire lifecycle |
| `ModuleHelper` | TaleWorlds.ModuleManager | Module loading and dependency resolution |
