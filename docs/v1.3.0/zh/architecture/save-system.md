# 存档系统 / Save System

**Version**: v1.3.0

## 概述 / Overview

v1.3.0 的存档系统通过 TaleWorlds.SaveSystem 命名空间实现，提供游戏数据的序列化和反序列化功能。

The save system in v1.3.0 is implemented through the TaleWorlds.SaveSystem namespace, providing game data serialization and deserialization.

## 与 v1.3.15 的差异 / Differences from v1.3.15

- v1.3.0 **没有**独立的 TaleWorlds.SaveSystem 模块
- v1.3.0 does **NOT** have a separate TaleWorlds.SaveSystem module
- TaleWorlds.SaveSystem 命名空间存在于 bannerlord-1.3.0 中，但实现方式与 v1.3.15 不同
- TaleWorlds.SaveSystem namespace exists in bannerlord-1.3.0 but implementation differs from v1.3.15
- v1.3.15 将 TaleWorlds.SaveSystem 拆分为独立模块
- v1.3.15 split TaleWorlds.SaveSystem into a separate module
- 没有 AutoGeneratedSaveManager 类（v1.3.15 新增）
- No AutoGeneratedSaveManager class (added in v1.3.15)
- 更简单的保存加载流程
- Simpler save/load process
- 缺少一些 v1.3.15 中的高级保存功能
- Missing some advanced save features from v1.3.15

## 核心类型 / Core Types

| Type | Description |
|------|-------------|
| Saveable | 属性标记，可序列化对象的属性 / Attribute mark for serializable object properties |
| SaveableProperty | 属性标记，用于标记可保存的属性 / Attribute mark for saveable properties |
| CampaignObjectBase | 战役对象的基类 / Base class for campaign objects |
| MBSaveLoad | 保存加载管理 / Save load management |

## 保存属性 / Save Attributes

使用 `SaveableProperty` 属性标记需要保存的属性:

Use `SaveableProperty` attribute to mark properties that need to be saved:

```csharp
[SaveableProperty(1)]
public Settlement Settlement { get; private set; }

[SaveableProperty(2)]
public MobileParty MobileParty { get; private set; }
```

## 存档流程 / Save Process

1. 游戏调用 MBSaveLoad 进行存档
2. CampaignTime 和其他基础类型被序列化
3. 各个对象通过 AutoGeneratedStaticCollectObjects 方法收集可保存数据
4. 数据被写入存档文件

1. Game calls MBSaveLoad to save
2. CampaignTime and other base types are serialized
3. Objects collect saveable data through AutoGeneratedStaticCollectObjects method
4. Data is written to save file

## 代码示例 / Code Example

```csharp
// In v1.3.0, save system is simpler
// Save occurs through MBSaveLoad
MBSaveLoad.SaveGame("save_name");

// Load game
MBSaveLoad.LoadGame("save_name");
```

## 注意事项 / Notes

- v1.3.0 的保存系统比 v1.3.15 简单
- v1.3.0 save system is simpler than v1.3.15
- TaleWorlds.SaveSystem 命名空间仍然可用，但组织方式不同
- TaleWorlds.SaveSystem namespace is still available but organized differently
- 不存在 AutoGeneratedSaveManager（这是 v1.3.15 的新特性）
- AutoGeneratedSaveManager does not exist (this is a v1.3.15 feature)
- 存档文件格式可能与 v1.3.15 不兼容
- Save file format may not be compatible with v1.3.15
