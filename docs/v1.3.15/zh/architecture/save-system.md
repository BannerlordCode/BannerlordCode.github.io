# 存档系统 / Save System

**Namespace**: TaleWorlds.SaveSystem
**Version**: 1.3.15 (NEW - 不存在于1.3.0)

## 概述 / Overview

TaleWorlds.SaveSystem 是 1.3.15 版本新增的模块化存档系统。在 1.3.0 中，存档逻辑分散在各个模块中，而 1.3.15 将其统一到一个专门的系统中。

TaleWorlds.SaveSystem is a new modular save system added in version 1.3.15. In 1.3.0, save logic was scattered across modules, but 1.3.15 unified it into a dedicated system.

## 核心类 / Core Classes

### SaveManager

```csharp
public static class SaveManager
{
    // 初始化全局定义上下文
    public static void InitializeGlobalDefinitionContext()
    {
        _definitionContext = new DefinitionContext();
        _definitionContext.FillWithCurrentTypes();
    }
    
    // 保存游戏
    public static SaveOutput Save(
        object target,           // 要保存的对象
        MetaData metaData,       // 元数据
        string saveName,         // 存档名
        ISaveDriver driver)      // 保存驱动
    {
        _isLoading = false;
        ISaveContext saveContext = new SaveContext(_definitionContext);
        
        if (saveContext.Save(target, metaData, out string error))
        {
            // 异步保存
            Task<SaveResultWithMessage> task = driver.Save(
                saveName, 1, metaData, saveContext.SaveData);
            // ...
        }
    }
    
    // 加载游戏
    public static LoadResult Load(string saveName, ISaveDriver driver)
    {
        return Load(saveName, driver, false);
    }
    
    public static LoadResult Load(
        string saveName, 
        ISaveDriver driver, 
        bool loadAsLateInitialize)
    {
        _isLoading = true;
        DefinitionContext definitionContext = new DefinitionContext();
        definitionContext.FillWithCurrentTypes();
        
        LoadData loadData = driver.Load(saveName);
        LoadContext loadContext = new LoadContext(definitionContext, driver);
        
        if (loadContext.Load(loadData, loadAsLateInitialize))
        {
            return LoadResult.CreateSuccessful(
                loadContext.RootObject, 
                loadData.MetaData, 
                loadCallbackInitializator);
        }
    }
}
```

### GameData

存档的实际数据载体：

```csharp
public class GameData
{
    public byte[] Header { get; internal set; }      // 头信息
    public byte[] Strings { get; internal set; }    // 字符串数据
    public byte[][] ObjectData { get; internal set; }   // 对象数据
    public byte[][] ContainerData { get; internal set; } // 容器数据
    
    public int TotalSize => Header.Length + Strings.Length 
        + ObjectData.Sum(d => d.Length) 
        + ContainerData.Sum(d => d.Length);
}
```

## 存档流程 / Save Flow

```
存档触发 (用户在菜单点击"保存")
    │
    ▼
Module.OnGameEnd()
    │
    ▼
Game.OnFinalize()
    │
    ▼
SaveManager.Save(target, metaData, saveName, driver)
    │
    ├─► 创建SaveContext
    │
    ├─► DefinitionContext 检查类型定义
    │
    ├─► SaveContext.Save() 序列化对象
    │       │
    │       ├─► 遍历所有[SaveableField]/[SaveableProperty]
    │       │
    │       ├─► 调用对应Serializer
    │       │
    │       └─► 生成GameData (Header + Strings + ObjectData + ContainerData)
    │
    └─► ISaveDriver.Save() 写入文件
            │
            └─► AsyncFileSaveDriver → ZIP压缩 → 写入.sav文件
```

## 加载流程 / Load Flow

```
读档触发 (用户在菜单点击"继续游戏"或选择存档)
    │
    ▼

    │
    ├─► 创建新的DefinitionContext
    │
    ├─► ISaveDriver.Load() 读取.sav文件
    │       │
    │       └─► 解压 → GameData
    │
    ├─► 创建LoadContext
    │
    ├─► LoadContext.Load() 反序列化
    │       │
    │       ├─► 读取Header重建对象结构
    │       │
    │       ├─► 读取ObjectData/ContainerData
    │       │
    │       └─► 解析字符串表
    │
    └─► 返回LoadResult
            │
            ├─► RootObject (重建的游戏对象)
            ├─► MetaData (存档元信息)
            └─► LoadCallbackInitializator (后期初始化回调)
```

## 存档文件格式 / File Format

存档文件 (.sav) 本质上是ZIP压缩包，包含：

```
save.zip
├── header.bin      # 二进制头
├── strings.bin     # 字符串表
├── objects_0.bin   # 对象数据块
├── objects_1.bin   # 更多对象数据
├── container_0.bin # 容器数据块
└── container_1.bin # 更多容器数据
```

## 使用示例 / Usage Example

```csharp
// 定义可存档类
[SaveableRootClass(5000)]
public class MyGameData
{
    [SaveableField(1)]
    public int Score;
    
    [SaveableField(2)]
    public string PlayerName;
    
    [SaveableProperty(3)]
    public List<Item> Inventory { get; set; }
}

// 保存游戏
public void SaveGame(string saveName)
{
    MyGameData data = new MyGameData { Score = 100, PlayerName = "Hero" };
    MetaData meta = new MetaData 
    { 
        SaveName = saveName,
        GameVersion = ApplicationVersion.FromParametersFile(null)
    };
    
    ISaveDriver driver = new AsyncFileSaveDriver();
    SaveOutput result = SaveManager.Save(data, meta, saveName, driver);
    
    if (result.Succeeded)
    {
        Debug.Print("Game saved successfully!");
    }
}

// 加载游戏
public void LoadGame(string saveName)
{
    ISaveDriver driver = new AsyncFileSaveDriver();
    LoadResult result = SaveManager.Load(saveName, driver);
    
    if (result.Succeeded)
    {
        MyGameData data = result.RootObject as MyGameData;
        Debug.Print($"Loaded: {data.PlayerName}, Score: {data.Score}");
    }
}
```

## Save Flow

```
User clicks "Save"
    │
    ▼
SaveManager.Save() → SaveContext.Save()
    │
    ▼
DefinitionContext checks types
    │
    ▼
GameData created (Header + Strings + ObjectData + ContainerData)
    │
    ▼
ISaveDriver.Save() → AsyncFileSaveDriver → ZIP file
```

## Load Flow

```
User clicks "Load"
    │
    ▼
SaveManager.Load() → ISaveDriver.Load()
    │
    ▼
GameData extracted from ZIP
    │
    ▼
LoadContext.Load() deserializes
    │
    ▼
LoadResult with RootObject returned
```
