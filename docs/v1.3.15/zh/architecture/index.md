# 架构概述 / Architecture Overview

**Namespace**: TaleWorlds.*
**Version**: 1.3.15 (compared with 1.3.0)

## 模块层次结构 / Module Hierarchy

Bannerlord 由 47 个 TaleWorlds.* 模块组成，形成以下依赖层次：

```
TaleWorlds.Library (基础工具)
    │
    ├── TaleWorlds.LinQuick (LINQ优化)
    ├── TaleWorlds.Localization (本地化)
    └── TaleWorlds.InputSystem (输入系统)
            │
            ▼
TaleWorlds.Core (核心：物品、角色、制作)
    │
    ├── TaleWorlds.ObjectSystem (XML反序列化)
    ├── TaleWorlds.DotNet (.NET互操作)
    └── TaleWorlds.ScreenSystem (屏幕管理)
            │
            ▼
TaleWorlds.Engine (原生引擎桥接)
    │
    ├── TaleWorlds.Engine.AutoGenerated
    └── TaleWorlds.NavigationSystem (导航系统)
            │
            ▼
TaleWorlds.MountAndBlade (游戏引擎核心)
    │
    ├── TaleWorlds.CampaignSystem (战役系统)
    ├── TaleWorlds.MountAndBlade.ViewModelCollection
    └── TaleWorlds.MountAndBlade.AutoGenerated
            │
            ▼
TaleWorlds.ModuleManager (模块管理)
TaleWorlds.SaveSystem (存档系统) ← 1.3.15 新增
```

## Module Hierarchy

Bannerlord consists of 47 TaleWorlds.* modules forming this dependency hierarchy:

```
TaleWorlds.Library (Base utilities)
    ├── TaleWorlds.LinQuick (LINQ optimizations)
    ├── TaleWorlds.Localization (Localization)
    └── TaleWorlds.InputSystem (Input handling)
            │
            ▼
TaleWorlds.Core (Core: items, characters, crafting)
    ├── TaleWorlds.ObjectSystem (XML deserialization)
    ├── TaleWorlds.DotNet (.NET interop)
    └── TaleWorlds.ScreenSystem (Screen management)
            │
            ▼
TaleWorlds.Engine (Native engine bridge)
    ├── TaleWorlds.Engine.AutoGenerated
    └── TaleWorlds.NavigationSystem (Navigation)
            │
            ▼
TaleWorlds.MountAndBlade (Game engine core)
    ├── TaleWorlds.CampaignSystem (Campaign system)
    ├── TaleWorlds.MountAndBlade.ViewModelCollection
    └── TaleWorlds.MountAndBlade.AutoGenerated
            │
            ▼
TaleWorlds.ModuleManager (Module management)
TaleWorlds.SaveSystem (Save system) ← NEW in 1.3.15
```
