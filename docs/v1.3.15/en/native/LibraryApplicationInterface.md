# LibraryApplicationInterface

**File**: `TaleWorlds.DotNet/LibraryApplicationInterface.cs`  
**Namespace**: `TaleWorlds.DotNet`

`LibraryApplicationInterface` is the static registry for native interfaces, holding singleton instances of all `INative*` interfaces.

## Class Definition

```csharp
internal class LibraryApplicationInterface
{
    internal static IManaged IManaged;
    internal static ITelemetry ITelemetry;
    internal static ILibrarySizeChecker ILibrarySizeChecker;
    internal static INativeArray INativeArray;
    internal static INativeObjectArray INativeObjectArray;
    internal static INativeStringHelper INativeStringHelper;
    internal static INativeString INativeString;
    
    private static Dictionary<string, object> _objects;
    
    internal static void SetObjects(Dictionary<string, object> objects);
```

## Important Notes

1. **Static readonly** - All `INative*` interfaces are static fields, valid for entire process lifetime
2. **Lazy initialization** - `SetObjects` called by native code at startup
3. **Namespace matching** - Dictionary keys use `TaleWorlds.DotNet.{InterfaceName}` format
4. **AutoGenerated** - `ScriptingInterfaceObjects` is auto-generated by tools
