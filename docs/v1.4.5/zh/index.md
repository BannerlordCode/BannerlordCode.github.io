# Bannerlord v1.4.5

::: tip 源码已就绪 / Source now available
v1.4.5 反编译源码已整理在 `bannerlord-1.4.5/Bannerlord.Source/`：玩法模块（SandBox / StoryMode / Multiplayer / CustomBattle / FastMode / BirthAndDeath / Native 扩展）共 2361 个 .cs 文件 / 2523 个类型；核心 `TaleWorlds.*` 程序集以 DLL 形式位于 `bin/`。完整类参考将逐步添加。
:::

## 模块清单

v1.4.5 共 **65 个 `TaleWorlds.*` 托管程序集**。相对 v1.3.15：

- **新增 14 个**：`CampaignSystem.FastMode`、`CampaignSystem.ViewModelCollection.BirthAndDeath`、`MountAndBlade.CustomBattle`、`MountAndBlade.DedicatedCustomServer.ClientHelper`、`MountAndBlade.GauntletUI`、`MountAndBlade.GauntletUI.AutoGenerated.0/.1`、`MountAndBlade.Multiplayer`(+`.GauntletUI`/`.View`/`.ViewModelCollection`/...)、`MountAndBlade.Platform.PC`、`MountAndBlade.View`
- **移除 4 个**：`MountAndBlade.GauntletUI.CodeGenerator`、`MountAndBlade.Launcher`、`MountAndBlade.SaveSystem.CodeGenerator`、`MountAndBlade.SteamWorkshop`
- **玩法模块回归**：v1.3.15 源码包缺失的 `SandBox`(+.GauntletUI/.View/.ViewModelCollection) 与 `StoryMode`(+...) 在 1.4.5 中以源码形式提供。

详见 v1.3.15 的 [版本差异](../../v1.3.15/zh/architecture/version-delta) 与 [SDK 总览](../../v1.3.15/zh/architecture/sdk-overview)。

## 源码结构

```
bannerlord-1.4.5/Bannerlord.Source/

```
